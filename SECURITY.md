# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch and the most recent published release.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities privately by contacting the maintainers at:

- 10243652+IgnaceMaes@users.noreply.github.com

Include:

- affected version/commit
- reproduction steps or proof-of-concept
- impact assessment

We will acknowledge receipt and provide follow-up as quickly as possible.
