## What this package provides

- JSON:API-style resource route generation on top of Django Ninja
- Querystring parsing for `filter`, `sort`, `include`, `fields`, and `page`
- JSON:API-style exception payloads
- Django ORM-based data layer baseline
- Atomic operations route wiring (`/operations`)

For a quick first integration, start with [Quickstart](quickstart.md).
